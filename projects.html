Free Online Digital Snooker Scoreboard

JavaScript CSS3 HTML5 Mobile-First UX Progressive Web App

An interactive, mobile-responsive telemetry tool engineered to track frame scores, consecutive break accumulations, and foul penalties straight from the table edge. Built entirely serverless to eliminate infrastructure overhead and ensure instantaneous response times on remote networks.

I regularly play snooker and developed this tool to enhance my gameplay experience and provide a more engaging way to track matches. I may expand it in the future to include additional features and a backend cloud database.

If you're using a mobile device, you can add this tool to your home screen and use it offline.

Core Architecture Highlights: Implements real-time dynamic state-tracking algorithms to compute continuous frame runs, turn switching context, and opposite-player penalty routing upon foul calculation.
